These batch files convert Minecraft Texture packs to Minetest Texture packs.
============================================================================
License:WTFPL
Author:jojoa1997
Depends:Windows OS
============================================================================
Paste both files downloaded in the texture pack you want.
To use just double click Tpconvert.bat
Due to some changes in how minecraft does textures some textures might be different aka wrong and also some of the blocks are different from the textures.
For example gold is considered mese. Also the texture for torches are only for on the floor.
